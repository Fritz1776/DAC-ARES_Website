Dona Ana County AREDN Mesh

The DAC-ARES local mesh currently has four independent internet connections, including Starlink, cable internet, and two cellular networks.

AREDN access warning: The node links below are internal .local.mesh addresses. They will only work when your device is connected to an AREDN node that is connected to this mesh network. These links are not reachable from the normal public internet.

About AREDN

AREDN is an amateur-radio mesh networking system used to create resilient IP-based communications over RF links. DAC-ARES can use AREDN to support local services such as Winlink access, file sharing, situational awareness, messaging, and emergency communications support when normal infrastructure is degraded.
