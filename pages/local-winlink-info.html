Local Winlink Information

NM5XD-10 Packet Winlink Gateway

Local packet Winlink access for Dona Ana County ARES and the surrounding area.

Gateway Overview

DAC-ARES operates packet Winlink gateway NM5XD-10 on the statewide packet frequency of 145.010 MHz.

  • When normal internet service is available, the gateway can support normal Winlink traffic flow.
  • When the internet goes down, this gateway can continue handling local Winlink traffic through a local Winlink Post Office.
  • The system can also relay Winlink traffic over HF when internet-based routing is unavailable.
  • This gateway and its Post Office remain fully operational on the AREDN network with or without the internet.
  • Winlink clients on AREDN-connected hardware can telnet into the gateway across the mesh network.
Operational note: This setup is intended to preserve local and regional message capability during internet outages, while also allowing continued integration with AREDN-connected stations and services.

Key Functions

What this local Winlink system provides during normal and degraded conditions.

Packet Gateway

Callsign: NM5XD-10
Frequency: 145.010 MHz

Local Post Office

Supports local message handling when internet service is unavailable.

HF Relay Capability

Allows Winlink traffic to be relayed over HF when needed.

AREDN Access

Accessible from AREDN-connected hardware by telnet across the local mesh.

Use on the AREDN Network

Local mesh-connected access to Winlink services.

  • The Winlink gateway and Post Office are available across the AREDN network, whether or not the internet is up.
  • Stations using AREDN-connected computers or equipment can access the system by telnet over the mesh.
  • This provides an additional path for message handling, especially when conventional internet access is degraded or unavailable.

AREDN Telnet Access

Host: NM5XD-1.local.mesh
Port: 8772
This service is intended for stations connected to the local AREDN mesh.
AREDN access note: The host name above is a .local.mesh address. It will only work when your device is connected to the AREDN network.
